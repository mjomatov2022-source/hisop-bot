
## 2-QADAM: Railway.app da joylash (BEPUL)

1. railway.app ga boring
2. GitHub bilan ro'yxatdan o'ting
3. "New Project" → "Deploy from GitHub repo"
4. bot.py va requirements.txt ni yuklang
5. Variables bo'limiga BOT_TOKEN va GROQ_API_KEY ni kiriting
6. Deploy bosing
